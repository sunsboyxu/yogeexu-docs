# Chrome

## Mac chrome 快捷键

macOS 下 Chrome 快捷键使用

``` js
command + W // 关闭当期网页
command + T // 打开新的标签页
shift + command + F // 全屏浏览器
shift + command + B // 隐藏标签
```
