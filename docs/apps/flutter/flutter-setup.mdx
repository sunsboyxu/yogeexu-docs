## Windows 环境配置

相关参考

- [参考 1](https://cloud.tencent.com/developer/article/2368655)

- 1、下载 [Flutter SDK](https://docs.flutter.dev/release/archive?tab=windows)，解压到平时安装软件的目录，如：D:\flutter
- 2、在 _环境变量_ -> 系统环境变量 -> **Path** 中，添加 D:\flutter\bin
- 3、打开终端，输入：flutter --version
- 4、然后输入：flutter doctor 进行 flutter 开发环境的检测
