# pnpm

- [pnpm](https://pnpm.io/zh/)

为什么叫pnpm？是因为pnpm作者对现有的包管理工具，尤其是npm和yarn的性能比较特别失望，所以起名叫做perfomance npm，即pnpm（高性能npm）
