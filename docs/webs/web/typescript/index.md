# Typescript

## 安装

``` bash
npm install typescript -g
```

``` js
tsc --version
```

``` js
tsc --init
```

## 基于 webpack5 搭建一个 ts 项目

项目名称：ts-demo

``` bash
npm init -y
tsc --init
```

新建 src、public

``` bash
npm install webpack webpack-cli webpack-dev-server ts-loader html-webpack-plugin
```

## 基于 vue-cli vue2.x 版本 未使用 ts 的项目

``` bash
vue add @vue/typescript
```

## TS特点

- 类型注解、类型检测
- 类
- 接口
- 泛型
- 装饰器
- 类型声明
