# Umijs

[Umijs中文网](https://umijs.org/zh-CN)

## 环境要求

node v10.13 以上

## 安装项目

- 新建一个项目 umi-app1
- 命令窗口进入目录执行：yarn create @umi-js/umi-app
- 命令窗口进入目录执行：npx @umijs/create-umi-app

## 使用命令快创建页面

- npx umi g page profile/index --typescript --less

## 网络请求工具

[umi-request](https://github.com/umijs/umi-request)
