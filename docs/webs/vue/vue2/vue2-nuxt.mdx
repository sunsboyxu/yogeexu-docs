## 安装项目

<Tabs>
<TabItem value="yarn" label="yarn">

```bash
yarn create nuxt-app 项目名称
```

</TabItem>
<TabItem label="npx" value="npx">

```npx
npx create-nuxt-app 项目名称
```

</TabItem>
<TabItem label='npm' value='npm'>

```bash
npm init
```

</TabItem>
</Tabs>

根据提示选择一些配置，如：eslint + prettier 等

当前使用的 nuxt 版本：2.15.8
